### Installation

<div id="installation"></div>

1. Clone the repo

   ```sh
   git clone git@github.com:hosein-fatthi/itshosein.git
   ```

2. Install NPM packages
   ```sh
   npm install
   ```
3. For development

   ```js
     npm run dev
   ```

   For production

   ```js
     npm run build

     npm start
   ```
